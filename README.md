# MyClarinet

Бесплатное приложение для изучения аппликатуры кларнета.
Приложение можно скачать и использовать локально — достаточно распаковать архив и открыть файл в любом современном браузере.
Приложение является бесплатным и может использоваться как в личных, так и в коммерческих целях.
Указание авторства не требуется.
